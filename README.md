# Quantum Machine Learning Lab M0 – Hello World Example

This project introduces the fundamentals of Quantum Machine Learning (QML) using PennyLane and Python.

## Project Overview
The notebook explores basic quantum computing concepts and demonstrates how quantum circuits can approximate mathematical functions using Quantum Machine Learning techniques.

The project includes:
- Quantum circuit setup using PennyLane
- Mathematical function approximation
- Quantum node (QNode) implementation
- Visualization of quantum model outputs
- Comparison between target functions and QML predictions

Functions explored in this lab include:
- Square function
- Cosine function
- Polynomial functions
- Custom mathematical functions such as sin(x)

## Features
- Quantum Machine Learning workflow
- QNode implementation with PennyLane
- Function approximation using quantum circuits
- Data visualization and graph analysis
- Quantum model experimentation without optimizers
- Introduction to variational quantum circuits

## Technologies Used
- Python
- PennyLane
- NumPy
- Matplotlib

## Results
- Successfully implemented multiple mathematical functions using quantum circuits.
- Visualized quantum predictions against target mathematical functions.
- Demonstrated the ability of QML models to approximate nonlinear functions.
- Explored foundational concepts in quantum neural networks and variational quantum computing.

## Learning Outcomes
- Understanding basic Quantum Machine Learning concepts
- Building quantum circuits with PennyLane
- Implementing QNodes for mathematical modeling
- Visualizing QML predictions and function approximation
